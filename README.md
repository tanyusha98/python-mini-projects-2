# Python Mini Projects 2

This repository contains small Python projects with corresponding tests.

## Projects

1. **project1.py** - simple greeting function
2. **project2.py** - simple calculator function
3. **project3.py** - check if a number is even

## Structure

python-mini-projects-2/
│
├─ project1.py
├─ project2.py
├─ project3.py
├─ README.md
├─ requirements.txt
├─ .gitignore
└─ tests/
├─ test_project1.py
├─ test_project2.py
└─ test_project3.py

## How to run

1. Install requirements:
pip install -r requirements.txt


2. Run any project:


python project1.py
python project2.py
python project3.py


3. Run all tests:


pytest
